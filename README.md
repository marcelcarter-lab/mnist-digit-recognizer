# MNIST Digit Recognizer (Neural Network Project)

This project is a Python-based neural network model that recognizes handwritten digits from images.

It uses the MNIST dataset to train a model capable of predicting numbers from 0 to 9.

---

## Project Description

The goal of this project is to build a simple image classification system using a neural network that can:

- Read handwritten digits
- Process images
- Predict the correct number (0–9)

---

## Technologies Used

- Python
- NumPy
- TensorFlow / Keras (or PyTorch if you used it)
- Matplotlib
- MNIST dataset

---

## How It Works

1. Load and preprocess MNIST dataset
2. Build a neural network model
3. Train the model on labeled digit images
4. Test it on new handwritten images
5. Output predicted digit
